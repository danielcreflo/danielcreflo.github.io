About Me

Securing Systems.

I'm Daniel Creflo Mtonga, a BSc Computer Systems & Security graduate with a deep passion for understanding how systems work — and how to make them resilient against modern threats. My academic journey has taken me across cybersecurity, software development, network engineering, and digital forensics.

I enjoy tackling real-world security challenges, whether that means analysing malware behaviour, building secure web applications, configuring hardened network infrastructures, or investigating digital evidence trails. I believe security is not a feature — it's a mindset.

When I'm not in a terminal or code editor, I'm reading up on the latest threat intelligence, writing security write-ups, or exploring new tools in my home lab environment.
